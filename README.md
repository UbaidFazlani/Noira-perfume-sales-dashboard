# Noira Perfume Sales Dashboard

This project is an Excel-based dashboard created to analyze perfume sales performance.  
It helps understand sales trends, product performance, customer behavior, and regional distribution.

# Project Details
- The dashboard shows overall sales, order count, and average order value.
- It highlights top-selling perfumes and category-wise performance.
- Regional, customer gender, and sales channel analysis are also included.

# Dataset
- The dataset contains perfume sales data with fixed pricing, categories, ratings, and review counts.
- Sales data is structured properly with correct region mapping and consistent product information.

# Tools Used
- Microsoft Excel
- Pivot Tables and Pivot Charts
- Data cleaning and formatting

# Objective
The objective of this project is to analyze perfume sales data and identify key trends, high-performing products, and customer segments, using an interactive Excel dashboard to support better business and sales decisions.
